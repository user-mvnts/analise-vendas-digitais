# Análise de Vendas Digitais

Projeto de estudo focado na organização, tratamento e análise de dados de vendas digitais,
com o objetivo de gerar métricas confiáveis para apoio à tomada de decisão.

## Objetivo

Desenvolver um projeto completo de análise de dados, abrangendo:

- Organização e estruturação dos dados  
- Tratamento e limpeza das informações  
- Extração de métricas com SQL  
- Análise de dados com Python  
- Visualização de indicadores  

## Tecnologias Utilizadas

- **Python** — Análise, tratamento e exploração de dados  
- **SQL** — Extração e agregação de métricas  
- **Jupyter Notebook** — Ambiente de análise  
- **Git / GitHub** — Versionamento e portfólio  

## Estrutura do Projeto
```
analise-vendas-digitais
├── README.md
├── data
│ ├── raw # Dados brutos
│ └── processed # Dados tratados
├── notebooks # Análises em Python
├── sql # Consultas SQL
└── dashboard # Visualizações e métricas
```
